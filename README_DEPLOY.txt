Hướng dẫn nhanh để deploy lên GitHub Pages (tên repo: fuga-light.github.io)

1) Tạo repository public trên GitHub có tên EXACT: fuga-light.github.io
2) Upload tất cả nội dung trong thư mục này vào root của repository (index.html, chapter1.html, styles.css, script.js, assets/, qr.png)
3) Sau khi upload xong, GitHub Pages sẽ phục vụ trang tại:
   https://fuga-light.github.io
4) Để cập nhật chương mới: thêm file chapter2.html, sửa index.html để thêm link tới chương 2.
5) Để thay bìa: đặt ảnh dọc vào assets/cover.jpg và chỉnh index.html nếu cần.
6) QR: file qr.png trong gói là placeholder if QR generation failed; bạn có thể thay bằng QR code thật trỏ tới https://fuga-light.github.io

Nếu cần mình hỗ trợ upload, bạn có thể gửi invite collaborator hoặc upload zip và mình hướng dẫn tiếp.
